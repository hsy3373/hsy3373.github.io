# 인라인 코드

코드 블록Permalink
\```언어 이름(소문자)
이 부분에 코드 적기
\```

언어 이름까지 적어주면 예약어, 변수 등등에 색깔 하이라이팅이 들어간다. 예를 들어

```c++
#include <iostream>
using namespace std;

int main()
{
cout « “Hello World !” « endl; // 안녕!
}
```
