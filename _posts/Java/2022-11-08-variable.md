> ## Variable(변수)

### 1. 변수란

- 메모리(RAM)에 값을 기록하기 위한 공간
- 변수 사용시 가독성이 좋아지며 재사용성 증가로 인한 코드량 감소 -> 유지보수가 용이하다

### 2. 변수 선언

```java

자료형 변수명;
  int num;

```

위와 같이 앞에 자료형 뒤에 변수명을 입력하여 선언한다.

이때

```java
int num = 1;
```

위와같이 선언과 동시에 값을 넣어줄 수도 있는데 이를 초기화라고 한다.
